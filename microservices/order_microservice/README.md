# Order - naročanje e-vinjete

## Opis storitve
Uporabnik bo lahko naročil e-vinjeto za svoje vozilo. 

Zahtevane podatke:
- ime
- priimek
- e-mail
- registerska številka vozila
- datum začetka veljavnosti vinjete

Po uspešnem vnosu podatkov, se podatki zapišejo v MongoDB.
