# User - prijava na e-vinjete

## Opis storitve
Uporabnik se lahko prijavi v sistem in pridobi link za nadaljevanje z nakupom

Zahtevane podatke:
- e-mail

Po uspešnem vnosu podatkov, se podatki zapišejo v MongoDB.
